### GreasyFork的优化

* 在菜单按钮中增加录入登录账号/密码
* 自动登录账号
* 清空已录入的账号/密码
* 在脚本页面添加一个`寻找引用`，用于查询该脚本/库是否被其它的脚本调用
* 在脚本页面中的`今日安装`下面添加`今日检查`
* 美化页面（Markdown/Button/Radio/TextArea）
* 添加复制代码按钮
* 添加Markdown右上角复制按钮
* 源代码同步【脚本列表】
* 源代码同步【未上架的脚本】
* 源代码同步【库】
